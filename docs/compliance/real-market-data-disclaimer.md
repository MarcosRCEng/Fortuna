# Disclaimer de Dados Reais de Mercado

## Texto padrão

> Os dados de mercado exibidos podem ter atraso, inconsistências ou indisponibilidade temporária. O Fortuna é uma experiência educativa e simulada. As informações exibidas não constituem recomendação de investimento, consultoria financeira, oferta, análise individualizada ou promessa de rentabilidade.

## Locais onde deve aparecer

O disclaimer deve aparecer nos seguintes pontos quando dados reais estiverem habilitados ou forem mencionados:

- tela de mercado;
- detalhe do ativo;
- tela de ordem simulada;
- documentação;
- rodapé ou tooltip de dados reais;
- mensagens relacionadas à atualização de cotações;
- telas ou componentes que exibam preço real, variação real ou histórico real.

## Diretrizes de linguagem

A linguagem do Fortuna deve evitar:

- “compre”;
- “venda”;
- “melhor investimento”;
- “garantia de retorno”;
- “lucro certo”;
- “recomendado para você”;
- “oportunidade imperdível”.

Preferir linguagem educativa:

- “este ativo apresentou variação de mercado”;
- “simulação baseada em dados disponíveis”;
- “entenda o risco antes de decidir”;
- “dados podem estar atrasados”;
- “atividade apenas educativa”.

## Regras de produto

Mesmo com dados reais:

- o saldo continua fictício;
- as ordens continuam simuladas;
- a carteira continua educativa;
- não há execução em corretora;
- não há análise individualizada;
- não há recomendação financeira.
